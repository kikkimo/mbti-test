# MBTI 答题系统全面重构设计文档

## 概述

重构 MBTI 答题系统，解决以下问题：
1. 快速点击回答后，进度增加但答题数不增加
2. 底部快速跳转不能跳转到对应题目
3. 分页排版不合理，按钮太大，标题不清晰
4. 配色不温馨温暖

## 一、配色系统（暖色调）

| 用途 | 新颜色 | 旧颜色 | 说明 |
|------|--------|--------|------|
| 主色 Primary | `#E67E22` | `#7C3AED` | 温暖的橙色 |
| 次色 Secondary | `#D35400` | `#EC4899` | 深橙色/珊瑚色 |
| 强调色 Accent | `#F39C12` | `#06B6D4` | 金黄色 |
| 背景 Background | `#FFF8F0` | `#FDF4FF` | 米白色/奶油色 |
| 卡片背景 | `#FFFBF7` | - | 略深的米色 |
| 文字 Text | `#4A3728` | `#1F2937` | 深棕色 |

## 二、答题交互重构

### 2.1 防抖机制

**问题**：`handleSelect` 使用 `setTimeout` 300ms 后自动跳转，快速点击会创建多个定时器

**解决方案**：
- 使用 `useRef` 存储定时器 ID
- 选择新答案时立即清除旧定时器
- 添加 `isTransitioning` 状态防止重复触发

### 2.2 状态管理优化

重构 `useTestState` hook：
- 增加 `pendingTransition` 状态
- 添加 `selectAnswer` 方法统一处理选择逻辑
- 确保答案记录和页面跳转的原子性

## 三、快速跳转面板重构

### 3.1 分页标签

- 标签高度：`py-4` 确保文字完整显示
- 标签样式：`text-base font-bold`
- 每个标签显示：维度缩写 + 中文名称
  - `E-I 外向/内向`
  - `S-N 感觉/直觉`
  - `T-F 思考/情感`
  - `J-P 判断/知觉`
- 激活状态：橙色背景 + 白色文字

### 3.2 网格布局

- 每页：4行 × 9列 = 36个按钮
- 按钮样式：`aspect-[3/2]` 扁平矩形
- 按钮文字：`text-sm font-medium`
- 间距：`gap-1.5` 紧凑排列

### 3.3 按钮状态颜色

| 状态 | 颜色 | 说明 |
|------|------|------|
| 已回答 | `#27AE60` 暖绿色 | 用户已选择答案 |
| 未回答 | 浅灰色 | 用户未选择 |
| 当前题 | `#E67E22` 橙色 | 当前正在回答的题目 |

## 四、进度条优化

- 显示已答题数而非当前题号
- 颜色改为暖色渐变（橙色到金黄色）
- 添加数字变化动画效果

## 五、视觉细节

- 卡片阴影改为暖色调
- 按钮悬停效果更柔和
- 统一圆角 `rounded-xl`
- 分组标题区域充足高度
